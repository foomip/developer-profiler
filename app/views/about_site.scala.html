I have seen a few sites out there where people have set up their own personal websites. However I wanted this be more than just an "about me" site. I wanted it to be a place where I could experiment with the latest tools and technologies on the web.

Typically when building production sites you would stick to more stable and proven web stacks. Obviously the ability to support a project long term and the availability of skills is an important consideration when building applications.

Since this site is for my own personal use, I have artistic freedom to choose tools and technologies. You don't really understand something until you have built something with it, so really this is my testing ground for new tech as well as my own personal website.

Here is a summary of the tools and technologies used in this site:

  • Front end:
    • Javascript (Including EcmaScript 6 for webapp).
    • Jquery 2.1.4.
    • WebComponents, WebComponentsJS for browser compatibility and ExpandJS (0.9.3 Beta) as the web components framework.
    • CSS3 and SASS for all styling and transition animations.

  • Back end:
    • Play 2.4 as the server side Framework and Scala 2.11 as the main programming language.
    • Slick 3.0 as the rective functional database mapper and SQLite as the database engine.
